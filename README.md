# ExamManager v3

T3 Stack / Drizzle SaaS app for ExamManager (exam timer & management).

Built with Next.js, NextAuth, Drizzle ORM, Tailwind CSS, and Stripe.

## Related

- [exams-landing](https://github.com/ExamManager/exams-landing)
- [exams-app-v1](https://github.com/ExamManager/exams-app-v1)
- [exams-app-v2](https://github.com/ExamManager/exams-app-v2)

## Setup

1. Copy `.env.example` to `.env` and fill in values (see `src/env.js` for the schema).
2. Install and run:

```bash
yarn install
yarn db:push
yarn dev
```
